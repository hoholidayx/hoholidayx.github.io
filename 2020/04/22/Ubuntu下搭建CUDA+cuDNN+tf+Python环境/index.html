安装NVIDIA CUDA

版本

  • Ubuntu：16.0.4
  • Toolkit：10.0
  • cuDNN：7.5.1 (April 22, 2019) for CUDA 10.0
  1. 官网下载CUDA Toolkit

安装cuda之前需要先安装gcc、g++和 build-essential等。

sudo apt-get install build-essential gcc g++ make binutils linux-headers-`uname -r`

运行安装脚本即可

sudo sh cuda_10.0.130_410.48_linux.run

配置CUDA环境变量


export PATH=/usr/local/cuda-10.0/bin${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-10.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

source ~/.bashrc
  1. 下载cuDNN

  • dev包

  • runtime包

安装

dpkg -i libcudnn7-dev_7.5.1.10-1%2Bcuda10.0_amd64.deb
dpkg -i libcudnn7_7.5.1.10-1%2Bcuda10.0_amd64.deb

Python安装

源码安装

  1. 官网下载需要的版本
  2. 将源码解压缩，进入到目录内编译安装
./configure
make
make install

通过PPA源安装Python


sudo apt-get install software-properties-common

sudo add-apt-repository ppa:deadsnakes/ppa

sudo apt-get update

sudo apt-get install python3.6 python3-pip

python3.6 -m pip install [Package_to_install]

安装Tensorflow-gpu

按照上面的cuda和cuDNN版本，这里只能装1.14.0的，不然会提示与cuDNN版本不兼容。

python3.6 -m pip install tensorflow==1.14.0 tensorflow-gpu==1.14.0
